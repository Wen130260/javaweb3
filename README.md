JavaWeb 用户模糊查询系
一个基于 JSP + Servlet + JDBC 实现的用户信息模糊查询项目，支持通过用户名关键词模糊匹配数据库中的用户数据，并实时展示在前端页面。
✨ 项目简介
本项目实现了 Web 端用户信息查询功能：用户在页面输入框中输入任意字符（如 “张”），系统会自动匹配数据库中用户名包含该字符的所有用户，并将完整用户信息（编号、密码、邮箱、性别、注册日期等）以表格形式展示在页面中。

技术 / 工具	说明
JDK 1.8+	Java 运行环境
Apache Tomcat 9.x	Web 服务器
MySQL 5.7/8.0	数据库存储用户信息
Servlet 3.1	处理请求与响应
JSP	前端页面渲染
JDBC	数据库连接与操作
Maven	项目依赖管理

🎯 核心功能
模糊查询：输入用户名中的任意字符，即可查询数据库中所有用户名包含该字符的用户
数据展示：查询结果以表格形式清晰展示，包含用户完整信息
请求处理：通过 Servlet 处理表单提交请求，JDBC 连接数据库执行查询
字符集处理：统一 UTF-8 编码，解决中文乱码问题
